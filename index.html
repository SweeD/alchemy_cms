Alchemy. A modern CMS for human beings.

About

Alchemy is a fully featured Web-CMS which beautifully integrates into rails.

Intuitive User Interface for humans!

With it's modern App-like User Interface even non technical users can manage complex websites in an easy and intuitive way.

Powerfull image processing

You will never need an image processor any more. With the support for ImageMagick® Technology Alchemy renders images just the size the designer ment the images to be and caches them on disc.

Rails Version

Alchemy is not yet Rails 3 and Ruby 1.9.2 compatible. We strongly recommend Rails 2.3.10 and Ruby 1.8.7.

Install

Install via Rails template:

We have a fancy Rails template that does all the installation stuff for you. You can find it here: 

http://github.com/magiclabs/alchemy-rails-templates/

Just enter: 

rails _2.3.10_ -d mysql -m https://github.com/magiclabs/alchemy-rails-templates/raw/master/install_alchemy.rb YOUR_APP_NAME

into your terminal and follow the instructions displayed after the templates executes.
